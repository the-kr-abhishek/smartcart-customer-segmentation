# smartcart-customer-segmentation

## Overview

This project performs customer segmentation on SmartCart shopping data using Python and clustering techniques. It includes data cleaning, exploratory data analysis, feature processing, and model-based grouping of customers by purchase behavior.

## Tech Stack

Python, Jupyter Notebook, pandas, numpy, matplotlib, seaborn, scikit-learn

## Features

* Data preprocessing and EDA
* Customer behavior analysis
* Clustering-based segmentation
* Visual insights

## Run
Clone the repo, download and store the dataset in project folder and open SmartCart.ipynb in Jupyter Notebook, then run all cells.
